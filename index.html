Проект: "Карта погоды"

Описание:

Веб-старница с возможностью узнать актуальную погоду, а также проверить знания в географии.

Принцип работы:

При нажатии на определенный город происходит API запрос на сервер openweathermap.org, через ключ и название города. Далее происходит преобразование ответа сервера в JSON(Объект). Затем получение данных из объекта и отображение их на странице, через пользовательски-удобный интерфейс(или нет).
